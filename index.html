Below is a demonstration to invoke the plugin, and specify whether you want it to refer inline content or via ajax.

  • This is a inline content --> Click me to refer to a inline content in a page, it can be hidden or visible(as you wish)

  • Ajax content --> For Ajax content remember to put a wrapper container around it to avoid content flow issues.

And simply invoke the code as

$('.overlay-init').on('click', function() { var $this = $(this); $this.overlay(); });

or with some options as

    debug: false,
    closeBtn: '.close',
    outerWrapper: '.outer-overlay',
    innerWrapper: '.inner-overlay',
    loadingClass: '.loading',
    loadingText: 'loading...',
    closeonEscape: true,
    colors: {
    closeButton: "#fff",
    background: "#222",
    loading: "#fff",
    opacity: '0.7'
    },
    callback: function() {
    },
//overlay can also be invoked by any of the following methods or by data-ajax/data-inline in html
    inline: "#some-id", //if using inline, mention the div id or any tag you like
    ajax: "file-url" //if using ajax, mention the url, default method is ajax
